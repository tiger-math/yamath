  htmlとは？
どうもこんにちは。
今回はhtml入門というのをやっていきます。

今回扱う html とは、一体どういうものなのかを説明します。

html とは、ウェブサイトの構造を作るために使うプログラム(コード)のことです。

実は、このサイトも、他の色んなサイトも html で構成されています。
実際、html 以外にも css や javascript なども一緒に使われることが多いのですが。

そこで今回は、html に着目して説明をする、という感じです。

では、早速、まずは環境づくりから始めましょう！

  環境構築


　　  htmlファイルの準備

ではまず、テキストエディタを開いてください。
なければ、メモ帳でもOKです。
windowsの場合はそのままメモ帳という名前のアプリで、
chromebookの場合は <t> というアイコンのアプリです。
まぁ、書けて保存できたらいいです。

本格的にやりたい方は、 "Visual Studio Code" や "ATOM" というアプリもいいかもしれません。
これらは無料ですので、それを使いたい方はそれを調べてみてください。

そして、エディタを開いたら、保存をしてみてください。
メモ帳などで書いている方は拡張子(~ .~~ の部分)を".html"にしてください。
そしたら、名前は何でもいいです。
たとえば、test.htmlなんかです。

そしたら、保存したhtmlファイルをブラウザ(chrome など)で開いてください。(作ったファイルをダブルクリック)

まだ、何も表示されません。
それでは、次にコーディングをしていっていきましょう！

　　  少しコーディング

それでは、次のコードを早速htmlファイルに打ち込んでみてください。

<!DOCTYPE html>
<html>
 <head>
  <meta charest="UTF-8">
  <title>html入門</title>
 </head>
 <body>
  
 </body>
</html>

このコードを入力できたら、ファイルを保存して、ブラウザを更新してみてください。

しかし、これでも何も表示されません。
なぜでしょうか。

それは、bodyタグ(<body> と </body> で囲まれている部分)の中に何も入力されてないからです。
しかし、headタグ の中の titleタグの中に html入門 と書いてあるので、タブのところに html入門 と表示されています。

ちなみにですけど、 titleタグ の中に入れた言葉がタブに入る言葉になります。

それでは、bodyタグの中に Hello World と入力して保存、ブラウザ更新してみてください。
すると、ブラウザに Hello World と表示されているのがわかります。

では、次でhtmlで色々なものを表示させてみましょう！

 いろんなタグ


　　 pタグ

では最初に紹介するのは、、、 ﾌﾟﾙﾙﾙﾙﾙﾙﾙﾙﾙﾙ ﾌﾟｺｰ　pタグ です。　ｲｪｰｰｰｲ

この p は段落を表す Paragraph の頭文字です。
この pタグ(<p> ~ </p>) で "Hello World" を囲んでみてください。

プログラム例.
<!DOCTYPE html>
<html>
 <head>
  <meta charest="UTF-8">
  <title>pタグ</title>
 </head>
 <body>
  <p>Hello World</p>
 </head>
</html>
すると、段落が形成されて、このようになります。
pタグ

Hello World



　　 hタグ

次は hタグ です！
この h は見出しを表す heading の頭文字です。

この hタグ は pタグ と同じように文字を表示するタグなのですが、
hタグ は大きさを変えることができます。

大きさの変え方は、 h の次に数字をつけるだけです。
この数字は 1~6 までの整数で、
数字が小さいほど、文字の大きさは大きくなります。

では試しに、 h1で囲んでみましょう！
bodyタグの中にこのように打ってみてください。
<h1>h1タグ</h1>

すると、下のような感じになると思います。
h1タグ

 h1タグ


では、次は、 h2 でやってみましょう！
h1のときと同じように、このように打ってみてください
<h2>h2タグ</h2>

すると、下のような感じになると思います。
h2タグ

　　 aタグ

次のタグはこちら！　aタグ
こちらのタグは Anchor の頭文字を取ったもので、リンクの出発点や到達点を指定するタグです。

では、bodyタグ の中にこのように打ってみてください。
<a href="https://google.com">google</a>

すると、このように表示されると思います。
aタグ
google


aタグに囲まれている文字列が表示され、その文字列をクリックすると、aタグの中の href という属性のサイトなどにとぶことができます。
では試しに、href= ~~ のところを下のようにに変えてみてください。
<a href="https://tiger-math.github.io/yamath">yamath</a>
すると、このyamathのホームページにとぶことができます。

更に、targetという属性を足して、下のようにすると、
<a href="https://tiger-math.github.io/yamath" target="_blank">yamath</a>

別のタブでそのリンクを開いてくれます。
aタグ-about blank
yamath

実はこのサイトの目次や、押すとホームページにとぶ画像にも aタグ を使用しています。
便利なタグですね。

　　 imgタグ

次はこちら!!　imgタグ　です。
こちらは、画像を表示させたいときに使うタグです。

では、 bodyタグ の中に、このように打ってみてください。
<img src="https://tiger-math.github.io/yamath/logo1.svg" alt="yamathのロゴ" height="100">

これをブラウザで実行すると、
imgタグ

上のような画像が出ます。

このタグは終了タグ(</~~>)がないので、 </img> は書く必要はありません。

この、出てくる画像の調整方法ですが、
src="~~" の部分になんの画像を入れるのかを入力します。

今回はurlで指定しましたが、そのhtmlと同じファイルの中に入っている画像だったら、
urlではなく、名前を入力するだけで大丈夫です。(※拡張子も書きます)

そして、 height という属性で、高さを決められます。
同じように、 width という属性は、幅を決められます。
数字の単位は px です。

また、 alt という属性で、
その画像が発見されなかったとき、あるいは、文を音声で読み上げていくときに
なんと出力するかを決められます。

例えば、先程のyamathのロゴのurlを下のようになくしてみると、、、
<img alt="yamathのロゴ" height="100">

画像が見つからないはずなので、下のようにalt属性に書かれているメッセージが出ます。
imgタグ


　　 tableタグ

次は tableタグ です。
これは、表のタグです。

では早速、次のプログラムを打ってみてください

<table border>
 <tr>
  <td>1-1</td>
  <td>1-2</td>
 </tr>
 <tr>
  <td>2-1</td>
  <td>2-2</td>
 </tr>
 <tr>
  <td>3-1</td>
  <td>3-2</td>
 </tr>
</table>


すると、このような表ができると思います。

2-12-2

この tableタグ の中に入っている
trタグ が表の列を、
tdタグ が表の中身(セル)を記述しています。

当然、 trタグ、tdタグ の数を増やせば、
表が大きくなります。

また、tableタグの属性に、border というものがありますが、
これを外すと、表の周りの枠がなくなります。

<table>

　　 ol,ulタグ

ol,ulタグはリストを作るタグで、
olタグは番号付きリストを、
ulタグは箇条書きをしてくれます。

試しに、次のコードを打ってみてください

<h2>olタグ</h2>
<ol>
 <li>項目1</li>
 <li>項目2</li>
</ol>

<h2>ulタグ</h2>
<ul>
 <li>項目1</li>
 <li>項目2</li>
</ul>

すると、このようなリストができると思います。
ol,ulタグ

olタグ

     
  1. 項目1
    2.  
  2. 項目2

ulタグ

     
  • 項目1
    •  
  • 項目2

ol,ulタグの中で、liタグで項目を囲んで、リストを作ります。
